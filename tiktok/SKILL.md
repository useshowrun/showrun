# TikTok Agent Browser Skills

Scrape public data from TikTok: user profiles, videos, and hashtag/challenge pages.
No login required for public content.

## Prerequisites

### Node.js 22+
Check: `node --version`

### Install Dependencies
```bash
cd tiktok && npm install
```

## Available Skills

| Skill | Script | Description |
|-------|--------|-------------|
| [Profile](tiktok-profile/SKILL.md) | `tiktok-profile/scripts/tiktok-profile.mjs` | Fetch profile + ~35 recent videos |
| [Videos](tiktok-videos/SKILL.md) | `tiktok-videos/scripts/tiktok-videos.mjs` | Fetch user videos with pagination |
| [Hashtag](tiktok-hashtag/SKILL.md) | `tiktok-hashtag/scripts/tiktok-hashtag.mjs` | Hashtag metadata + trending videos |
| [Comments](tiktok-comments/SKILL.md) | `tiktok-comments/scripts/tiktok-comments.mjs` | Comments on a video with pagination |

## Typical Workflow

```
1. Fetch profile    →  node tiktok-profile/scripts/tiktok-profile.mjs natgeo
2. Get more videos  →  node tiktok-videos/scripts/tiktok-videos.mjs natgeo --cursor <cursor>
3. Explore hashtag  →  node tiktok-hashtag/scripts/tiktok-hashtag.mjs nature
4. Read comments    →  node tiktok-comments/scripts/tiktok-comments.mjs <videoUrl> --max 50
```

## How It Works

TikTok is a JavaScript-heavy SPA with aggressive anti-bot measures. The scraper works by:

1. **Using camoufox** — a Playwright wrapper with realistic browser fingerprinting
2. **Navigating the real site** — not calling APIs directly (they require signed tokens)
3. **Intercepting API responses** — the browser loads the page and makes API calls automatically;
   we intercept these responses to extract structured data
4. **Reading embedded JSON** — TikTok embeds initial data in `__UNIVERSAL_DATA_FOR_REHYDRATION__`
   script tags, which we parse for profile data

## Key API Endpoints (observed from browser)

| Endpoint | Purpose |
|----------|---------|
| `/api/post/item_list/` | User's video list (35 per page, cursor-based) |
| `/api/challenge/detail/` | Hashtag metadata |
| `/api/challenge/item_list/` | Hashtag videos |
| `/api/user/detail/` | User profile (requires signed params) |
| `/api/comment/list/` | Video comments (20 per page, cursor-based) |

These endpoints use `X-Bogus` and `X-Gnarly` signature params that are generated by TikTok's
obfuscated JavaScript. The scraper avoids calling them directly — it lets the browser generate
the proper requests and intercepts the responses.

## Output Format

All scripts write structured output to stdout as `RESULT:{json}`. Logs go to stderr.
Parse results by reading lines starting with `RESULT:` from stdout.

## Limitations

- Private accounts return no posts
- Age-restricted content requires login
- `playUrl` (video download URLs) expire quickly (minutes to hours)
- Rate limiting: avoid running multiple instances in parallel
- TikTok detects unusual patterns — always use headless camoufox (not plain Playwright)
