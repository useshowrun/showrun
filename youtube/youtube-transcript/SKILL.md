# YouTube Transcript Scraper

Extract full transcripts (captions/subtitles) from any public YouTube video.

## What It Does

- Fetches all available caption tracks (languages) for a video
- Downloads the transcript in structured format with timestamps
- Supports automatic (ASR) and manual captions
- Returns clean text or timed segments

## Usage

```bash
# Basic - auto-select English or first available track
node scripts/youtube-transcript.mjs <videoId|url>

# Specify language
node scripts/youtube-transcript.mjs <videoId|url> --lang en
node scripts/youtube-transcript.mjs <videoId|url> --lang es

# Plain text mode (no timestamps)
node scripts/youtube-transcript.mjs <videoId|url> --text-only

# List available languages
node scripts/youtube-transcript.mjs <videoId|url> --list-langs
```

## Examples

```bash
node scripts/youtube-transcript.mjs dQw4w9WgXcQ
node scripts/youtube-transcript.mjs "https://www.youtube.com/watch?v=jNQXAC9IVRw" --lang en
node scripts/youtube-transcript.mjs dQw4w9WgXcQ --list-langs
node scripts/youtube-transcript.mjs dQw4w9WgXcQ --text-only
```

## Notes

- Transcripts are fetched via the YouTube timedtext API (same as what you get from the "Open transcript" button on YouTube)
- The API requires browser session cookies — we use camoufox to get them
- Videos without captions return an error with code `NO_TRANSCRIPT`
- Auto-generated (ASR) captions are available for most videos
- Language codes follow BCP-47 (e.g., `en`, `en-US`, `es`, `fr`)
- Use `--list-langs` to see all available tracks before fetching
